# Domestic payment consents

In this section, we describe how to get started with domestic payment consents which enable domestic payments.

With a bank registration that supports payments, you
can [create and authorise a domestic payment consent](./create-and-authorise-domestic-payment-consent/README.md).

Once the domestic payment consent is authorised, you
can [confirm funds and make a payment](./confirm-funds-and-make-payment/README.md).
